Instructions

Intial Sign Up

The design of this application is meant to be simple and intuitive in design. For teachers and students the inital sign up process is similar.

  1. Go to quizitapp.azurewebsites.net/registration to register your account. Be sure to indicate if you are a teacher or a student. You will be then directed to your acount page.
  2. Once logged in you will be able to Take a Test if your a student or create tests and view submissions if it is a teachers account.

    3. Student's Add A Class

    In order to add students to a class, they must have the code to the class. Without this code they will not be able to join the class or access the tests even with the provided test id.

    1. Go to quizitapp.azurewebsites.net/Account to add a class to the student profile.

    2. Click the "Add Class" button located beneath the User Icon Image.

    3. Enter in the class ID. provided by the teacher. It should be in the following format 0000-00.

    4. Click on "Add Class". The class then should appear, where the button to add the class was.

      5. Teachers's Add A Class

      1. Go to quizitapp.azurewebsites.net/Account to add a class to the teacher profile. Click the "Add Class" button located beneath the User Icon Image.

      2. Enter in the class name.

      3. Click on "Add Class". The class then should appear.
